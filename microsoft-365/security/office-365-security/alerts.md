# <a name="alerts-in-the-microsoft-365-defender-portal"></a>Alerty w Microsoft 365 Defender wiadomości

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]

**Dotyczy**
- [Exchange Online Protection](exchange-online-protection-overview.md)
- [Microsoft Defender dla Office 365 plan 1 i plan 2](defender-for-office-365.md)

Za pomocą funkcji alertów w portalu Microsoft 365 Defender można wyświetlać alerty dla organizacji i zarządzać nimi, w tym zarządzać zaawansowanymi alertami w ramach przeglądu aplikacji [Microsoft Defender dla chmury](/cloud-app-security/what-is-cloud-app-security).

## <a name="how-to-get-to-the-alerts-features"></a>Jak uzyskać dostęp do funkcji alertów

Alerty są dostępne w portalu Microsoft 365 Defender na <https://security.microsoft.com> stronie **Alerty o & zdarzeniach**\>. Aby przejść bezpośrednio do strony **Alerty**, użyj .<https://security.microsoft.com/alerts>

## <a name="alerts-features"></a>Funkcje alertów

W poniższej tabeli opisano narzędzia dostępne na stronie **Alerty** .

<br>

****

|Narzędzie|Opis|
|---|---|
|[Zarządzanie alertami](../../compliance/create-activity-alerts.md)|Alerty dotyczące aktywności mogą wysyłać powiadomienia e-mail do siebie lub innych administratorów, gdy użytkownicy wykonują określone działania w Microsoft 365. Alerty aktywności są podobne do wyszukiwania zdarzeń w dzienniku inspekcji, tyle że w przypadku wystąpienia zdarzenia, dla którym utworzono alert, otrzymasz wiadomość e-mail.|
|[Zarządzanie alertami zaawansowanymi](/cloud-app-security/what-is-cloud-app-security)|Użyj funkcji **Zarządzanie alertami zaawansowanymi** w usłudze Microsoft Defender dla aplikacji w chmurze, aby skonfigurować zasady, które mogą powiadamiać Cię o podejrzanych i anomalalnych działaniach w Microsoft 365. Po schłodeniu alertu możesz zbadać potencjalnie problematyczne sytuacje i w razie potrzeby podjąć działania w celu rozwiązania problemów z zabezpieczeniami.|
|
