# <a name="microsoft-security-portals-and-admin-centers"></a>Portale zabezpieczeń firmy Microsoft i centra administracyjne

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

Chociaż [Microsoft 365 Defender](microsoft-365-defender.md#the-microsoft-365-defender-portal) to nowa strona do monitorowania zabezpieczeń tożsamości, danych, urządzeń i aplikacji oraz zarządzania nimi, musisz uzyskać dostęp do różnych portali dla określonych zadań specjalistycznych.

> [!TIP] 
> Aby uzyskać dostęp do różnych odpowiednich portali z Microsoft 365 Defender, wybierz pozycję **Więcej zasobów** w okienku nawigacji.

## <a name="security-portals"></a>Portale zabezpieczeń

Administratorzy i operatorzy zabezpieczeń mogą przejść do następujących portali, aby zarządzać ustawieniami specyficznymi dla zabezpieczeń, badać możliwe działania związane z zagrożeniami, reagować na aktywne zagrożenia i współpracować z administratorami IT nad rozwiązywaniem problemów.
<p></p>

| Nazwa portalu | Opis | Link |
|---|---|---| 
| Microsoft 365 Defender portalu | Monitorowanie aktywności zagrożeń i reagowanie na nie oraz wzmacnianie stanu zabezpieczeń tożsamości, poczty e-mail, danych, punktów końcowych i aplikacji za pomocą usługi [Microsoft 365 Defender](microsoft-365-defender.md) | [security.microsoft.com](https://security.microsoft.com/) |
| Centrum zabezpieczeń usługi Microsoft Defender | Monitorowanie aktywności dotyczącej zagrożeń i reagowanie na nie w punktach końcowych przy użyciu funkcji programu [Microsoft Defender for Endpoint](/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-advanced-threat-protection) | [securitycenter.windows.com](https://securitycenter.microsoft.com/) |
| Centrum & zabezpieczeń | Zarządzanie [Exchange Online Protection](../office-365-security/exchange-online-protection-overview.md) i [programem Microsoft Defender dla usługi Office 365](/microsoft-365/security/office-365-security/defender-for-office-365) w celu ochrony poczty e-mail i usług współpracy oraz zapewnianie zgodności z różnymi przepisami dotyczącymi obsługi danych | [protection.office.com](https://protection.office.com) |
| Portal usługi Defender dla chmury | Użyj [programu Microsoft Defender dla chmury](/azure/security-center/security-center-intro) , aby zwiększyć bezpieczeństwo centrów danych i hybrydowych obciążeń w chmurze | [portal.azure.com/#blade/Microsoft_Azure_Security](https://portal.azure.com/#blade/Microsoft_Azure_Security/SecurityMenuBlade/0) |
| Portal tożsamości usługi Microsoft Defender | Identyfikowanie, wykrywanie i badanie zaawansowanych zagrożeń, naruszonych tożsamości i złośliwych działań w ramach niejawnego programu testów przy użyciu sygnałów usługi Active Directory z usługą [Microsoft Defender for Identity](/azure-advanced-threat-protection/what-is-atp) | [portal.atp.azure.com](https://portal.atp.azure.com/) |
| Portal usługi Defender dla aplikacji w chmurze | Korzystanie [z usługi Microsoft Defender dla aplikacji w](/cloud-app-security/what-is-cloud-app-security) chmurze w celu uzyskania zaawansowanej widoczności, kontroli nad podróżą danych oraz zaawansowanej analizy w celu identyfikowania i zwalczania cyberataków w usługach w chmurze | [portal.cloudappsecurity.com](https://portal.cloudappsecurity.com/) |
| Microsoft Security Intelligence portalu | Uzyskiwanie aktualizacji analizy zabezpieczeń dla programu Microsoft Defender dla punktu końcowego, przesyłanie przykładów i eksplorowanie encyclopedia podszywają się pod zagrożenia | [microsoft.com/wdsi](https://microsoft.com/wdsi) |

## <a name="portals-for-other-workloads"></a>Portale dla innych obciążeń

Te portale nie są przeznaczone specjalnie do zarządzania zabezpieczeniami, ale obsługują różne obciążenia pracą i zadania, które mogą mieć wpływ na twoje bezpieczeństwo. Odwiedź te portale, aby zarządzać tożsamościami, uprawnieniami, ustawieniami urządzenia i zasadami obsługi danych.
<p></p>

| Nazwa portalu | Opis | Link | 
|---|---|---| 
| Azure Portal | Wyświetlanie wszystkich zasobów platformy [Azure i zarządzanie nimi](/azure/azure-resource-manager/management/overview)  | [portal.azure.com](https://portal.azure.com/) |
| Azure Active Directory portalu | Wyświetlanie i zarządzanie [Azure Active Directory](/azure/active-directory/fundamentals/active-directory-whatis) | [aad.portal.azure.com](https://aad.portal.azure.com/) |
| Centrum zgodności platformy Microsoft 365 | Zarządzanie zasadami obsługi danych i zapewnianie [zgodności z przepisami](/compliance/regulatory/offering-home) | [compliance.microsoft.com](https://compliance.microsoft.com/) |
| Centrum administracyjne platformy Microsoft 365 | Konfigurowanie Microsoft 365, zarządzanie rolami i licencjami oraz śledzenie aktualizacji Microsoft 365 usług | [admin.microsoft.com](https://go.microsoft.com/fwlink/p/?linkid=2166757) |
| Microsoft Endpoint Manager administracyjne | Zarządzaj [Microsoft Endpoint Manager](/mem/configmgr/) urządzeniami i zabezpieczaj je przy użyciu połączonych funkcji intune i Menedżer konfiguracji intune | [devicemanagement.microsoft.com](https://devicemanagement.microsoft.com/) |
| Microsoft Intune portalu | Wdrażanie [Microsoft Intune](/intune/fundamentals/what-is-intune) i monitorowanie zgodności urządzeń za pomocą funkcji | [endpoint.microsoft.com](https://endpoint.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/DevicesMenu/overview)
