# <a name="whats-new-in-microsoft-secure-score"></a>Co nowego w programie Microsoft Secure Score

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

Aby witryna Microsoft Secure Score lepiej reprezentacyjna zapewniała poziom zabezpieczeń, wprowadzone zmiany zostały wprowadzone. Aby dowiedzieć się więcej o planowanych zmianach, zobacz [Co będzie się dzieje w aplikacji Microsoft Secure Score?](microsoft-secure-score-whats-coming.md)

Bezpieczny wynik firmy Microsoft można znaleźć w https://security.microsoft.com/securescore portalu [Microsoft 365 Defender sieci.](microsoft-365-defender.md#the-microsoft-365-defender-portal)

## <a name="july-2021"></a>Lipiec 2021

### <a name="added-improvement-action-related-to-microsoft-teams"></a>Dodano akcję udoskonalania związaną z Microsoft Teams

- Ograniczanie pomijania poczekalni spotkania przez użytkowników korzystających z połączeń telefonicznych
- Ograniczanie możliwości kontrolowania przez uczestników zewnętrznych podczas Teams spotkania
- Ograniczanie użytkownikom anonimowym możliwości rozpoczynania Teams spotkań
- Wymaganie skonfigurowania pogawędek dla Teams spotkań
- Konfigurowanie użytkowników, którzy mogą być obecni na Teams spotkaniach

### <a name="added-improvement-action-related-to-microsoft-defender-for-endpoint"></a>Dodano akcję udoskonalania związaną z usługą Microsoft Defender dla punktu końcowego

- Naprawianie zbierania danych czujnika punktu końcowego programu Microsoft Defender dla systemu macOS
- Naprawianie programu Microsoft Defender dla komunikacji z punktem końcowym niepełnosprawnym w systemie macOS
- Ustaw minimalną długość hasła na 15 lub więcej znaków w systemie macOS
- Ustaw wartość "Wymuszaj historię haseł" na "24 lub więcej haseł" w systemie macOS
- Ustaw "Maksymalny wiek hasła" na "90 lub mniej dni, ale nie 0" w systemie macOS
- Ustaw próg blokady konta na wartość 5 lub niższą w systemie macOS
- Włączanie Zapory w systemie macOS
- Enable Gatekeeper
- Włączanie ochrony integralności systemu (SIP)
- Włączanie szyfrowania dysku FileVault
- Ustawianie ekranu tak, aby był blokowany podczas uruchamiania wygaszacza ekranu w systemie macOS
- Upewnij się, że wygaszacz ekranu jest ustawiony tak, aby rozpoczynał się za 20 minut lub mniej w systemie macOS
- Zabezpieczanie folderów domu
- Włączanie ochrony Program antywirusowy Microsoft Defender czasie rzeczywistym dla systemu macOS
- Włączanie ochrony Program antywirusowy Microsoft Defender pua w trybie blokowania dla systemu macOS
- Włączanie Program antywirusowy Microsoft Defender w chmurze dla systemu macOS
- Aktualizowanie Program antywirusowy Microsoft Defender definicji systemu macOS
- Naprawianie programu Microsoft Defender dla zbierania danych czujnika punktu końcowego dla systemu Linux
- Naprawianie programu Microsoft Defender dla komunikacji z punktem końcowym niepełnosprawnym dla systemu Linux
- Unrestricted Access Accounts
- Włączanie ochrony Program antywirusowy Microsoft Defender czasie rzeczywistym dla systemu Linux
- Włączanie ochrony Program antywirusowy Microsoft Defender PUA w trybie blokowania dla systemu Linux
- Włączanie Program antywirusowy Microsoft Defender systemu Linux w chmurze
- Aktualizowanie Program antywirusowy Microsoft Defender dla systemu Linux

Microsoft Teams klientów zobaczą "Ograniczanie użytkownikom anonimowym możliwości dołączania do spotkań" jako nowe działanie udoskonalania w bezpiecznym wyniku.

## <a name="december-2020"></a>Grudzień 2020

### <a name="added-six-accounts-related-improvement-actions-for-microsoft-defender-for-endpoint"></a>Dodano sześć działań udoskonalania dotyczących kont dla programu Microsoft Defender dla punktu końcowego:

- Ustaw wartość "Minimalna długość hasła" na "14 lub więcej znaków"
- Ustaw wartość "Wymuszaj historię haseł" na wartość "24 lub więcej haseł".
- Ustaw "Maksymalny wiek hasła" na "60 lub mniej dni, ale nie 0"
- Ustaw "Minimalny wiek haseł" na "1 lub więcej dni".
- Wyłączanie wbudowanego konta administratora
- Wyłączanie wbudowanego konta gościa

## <a name="november-2020"></a>Listopad 2020

### <a name="removed-the-ability-to-create-servicenow-tickets-through-secure-score"></a>Usunięto możliwość tworzenia biletów ServiceNow za pośrednictwem bezpiecznego wyniku 

Możliwość tworzenia biletów serviceNow za pośrednictwem bezpiecznego wyniku przez przechodząc do > **ServiceNow** nie jest już dostępna. Dziękujemy za opinię i dalsze wsparcie podczas określania kolejnych kroków.

### <a name="added-three-services-related-improvement-actions-for-microsoft-defender-for-endpoint"></a>Dodano trzy działania udoskonalania dotyczące usług dla programu Microsoft Defender dla punktu końcowego:

- Naprawianie nieotwieranej ścieżki usługi dla Windows usług
- Zmienianie ścieżki wykonywalnego usługi do wspólnej chronionej lokalizacji
- Zmienianie konta usługi w celu uniknięcia hasła w pamięci podręcznej w rejestrze systemu Windows

## <a name="august-2020"></a>Sierpień 2020

### <a name="updated-improvement-action-for-azure-active-directory"></a>Zaktualizowano akcję udoskonalania dla Azure Active Directory

- Włączanie zasad w celu zablokowania starszego uwierzytelniania

## <a name="incompatibility-with-identity-secure-score"></a>Niezgodność z bezpiecznym wynikiem tożsamości

W ostatniej wersji programu Microsoft Secure Score opublikowano ulepszony model wyników. Te zmiany umożliwiają bardziej elastyczny i dokładny widok twojego wpisu w zabezpieczeniach. Jednak te aktualizacje tymczasowo niezgodno z wynikiem bezpiecznego działania programu Microsoft.

Z czasem bezpieczny wynik tożsamości przejmie nowy model oceniania. Do tego czasu klienci będą widzieć różnice w wynikach zgłoszonych przez program Microsoft Secure Score i wynik bezpiecznego tożsamości. Przepraszamy za wszelkie niedogodności powstałe na tym niedogodności i pracujemy nad zapewnieniem większej zgodności tych doświadczeń w przyszłości.

## <a name="updated-improvement-actions"></a>Zaktualizowane działania udoskonalania

- Dodano Azure Active Directory działania ulepszeń
- Dodano akcje udoskonalania usługi Microsoft Defender for Identity
- Pomoc techniczna dla programu Microsoft Defender w zakresie zagrożeń [w & zarządzania lukami w](/windows/security/threat-protection/microsoft-defender-atp/next-gen-threat-and-vuln-mgt) zabezpieczeniach
    - Wszystkie wydane zalecenia dotyczące zabezpieczeń udostępniane przez firmę TVM są teraz dostępne

## <a name="updated-interface-and-functionality"></a>Zaktualizowano interfejs i funkcje

* Wszystkie nowe widoki metryk i trendów dla ciso i dyskusji na poziomie potencjalnych pomysłów
* Nowe sposoby śledzenia i analizy wyników
* Lepsze śledzenie i lepsze zrozumienie regresji wyników
* Filtrowanie, tagowanie, wyszukiwanie i grupowanie akcji udoskonalania
* Zarządzaj swoimi przyszłymi celami przy użyciu projekcji wyników i planowanych działań
* I nie tylko!
